# Скрины реализации

Скриншоты работающего приложения Places Remember.

| Файл | Описание |
|------|----------|
| `главная страница.png` | Главная: приветствие, кнопки «Войти через VK» / «Войти через Google» и «Войти как тестовый пользователь» |
| `список воспоминаний.png` | После входа: имя, аватар (если есть), кнопки «Добавить воспоминание» и «Выйти», список мест или «У вас пока нет ни одного воспоминания» |
| `воспоминание.png` | Форма добавления: карта Leaflet (выбор точки), поля «Название места» и «Комментарий», кнопка «Сохранить» |

Для единообразия можно переименовать в латиницу: `main_page.png`, `memory_list.png`, `add_memory.png`.
