### 3. Places Remember

Веб-приложение для хранения впечатлений о посещаемых местах (Django).

**Ссылка:** [Places Remember](./Places%20Remember)

**Основные возможности:**
- Вход через VK или Google (OAuth2), тестовый вход без OAuth (DEBUG)
- Главная: приветствие для гостей, список воспоминаний для авторизованных
- Добавление воспоминания: карта Leaflet, название, комментарий
- Django 4.2, Bootstrap 5, SQLite/PostgreSQL
