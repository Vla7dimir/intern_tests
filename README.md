## Проекты



### 3. AppBooster — AB Testing API

REST API для проведения AB-тестов в мобильных приложениях.

**Ссылка:** [appBooster](./appBooster)

**Основные возможности:**
- Получение экспериментов для устройства по заголовку `Device-Token`
- Гарантия постоянства группы (одно устройство — одни и те же значения)
- Взвешенное распределение (button_color, price)
- Статистика по экспериментам (JSON и HTML)
- FastAPI, SQLAlchemy, SQLite/PostgreSQL, Pydantic
- Миграции Alembic, тесты pytest, Docker
- Структура проекта в стиле Python (папка `app/`, `config/`, `tests/`)
